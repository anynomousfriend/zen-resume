## âœ¨ Animations & Interactions

### 1. ASCII Blossoms
- **Animation**: Floating up and down, rotating, opacity changes
- **Duration**: 2-6 seconds per animation cycle
- **Effect**: Creates a dreamy, flowing atmosphere

### 2. Floating Petals
- **Animation**: Fall from top to bottom with rotation
- **Duration**: 8-18 seconds
- **Delay**: Random start times for natural effect

### 3. Hero Text
- **Animation**: Fade in + slide up
- **Stagger**: 100ms between each text element
- **Easing**: Elastic bounce effect

### 4. Feature Cards
- **Hover**: Scale to 1.02, stronger shadow
- **Scroll**: Scale from 0.8 to 1 when visible
- **Delay**: 150ms stagger between cards

### 5. Buttons
- **Hover**: Scale to 1.05, enhanced shadow
- **Press**: Slight scale down
- **Transition**: 300ms smooth

### 6. Form Inputs
- **Focus**: Pink ring, border color change
- **Hover**: Subtle border color shift
- **Transition**: 300ms smooth

### 7. Section Navigation
- **Active**: Pink gradient background, scale up
- **Inactive**: Transparent, hover for pink tint
- **Transition**: All properties animated

### 8. Theme Toggle
